# Public Assets

## Required Images

Please copy your images to this folder with these exact names:

1. **profile-main.jpg** - Your main profile photo (the one with glasses)
   - This will be used in the hero section

2. **profile-side.jpg** - Your secondary profile photo
   - This will be used in the "Who is Raihan?" section

## How to add the images:

1. Copy your image files to this `public` folder
2. Rename them exactly as shown above:
   - Main photo → `profile-main.jpg`
   - Secondary photo → `profile-side.jpg`

The website will automatically use these images once they are in place.
If the images are not found, it will show the "AJ" placeholder instead.
